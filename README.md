# Proyecto - Ingeniería de software

Prototipo de aplicación web diseñada para la presentación, estudio y esquematización
de datos relacionados a películas, destinado para usuarios de todo tipo.

Para ejecutar el proyecto:

1. Crear un archivo llamado auth.json (el archivo dispuesto para el proyecto se proporcionará de manera privada al docente), con la
información siguiente:
	dominio : URL del dominio creado en Auth0 para el módulo de inicio de sesión
		(más info en https://auth0.com/)
	client_id: Id de cliente proporcionado por la misma entidad mencionada anteriormente

2. Ejecutar run.cmd (o ejecutar el archivo index.js presente en la raiz del proyecto usando Node.js)
3. La página web por defecto podrá encontrarse ingresando en el navegador a localhost:3000
